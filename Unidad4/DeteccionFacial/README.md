# Proyecto Unidad 4 Detección Facial

En este proyecto desarrollamos un sistema de detección facial y reconocimiento de emociones mediante una red neuronal convolucional entrenada con el dataset FER2013, el sistema identifica en tiempo real emociones básicas como enojo, felicidad, neutral y tristeza, procesando el rostro captado por la cámara para ofrecer resultados rápidos y precisos.
